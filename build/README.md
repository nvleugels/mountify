# Build Assets

This directory contains application icons and build resources.

## Icons Needed

Please replace the placeholder SVG files with actual icon files:

### Required Icons

1. **icon.ico** - Main application icon (256x256)
   - Format: ICO file
   - Sizes: 16x16, 32x32, 48x48, 256x256

2. **tray-icon.png** - System tray icon (16x16 or 32x32)
   - Format: PNG file with transparency
   - Size: 16x16 for standard DPI, 32x32 for high DPI

## Creating Icons

You can use tools like:

- [IcoFX](https://icofx.ro/) - Icon editor for Windows
- [GIMP](https://www.gimp.org/) - Free image editor
- [ImageMagick](https://imagemagick.org/) - Command-line tool
- Online converters like [ConvertICO](https://convertico.com/)

## Icon Design Guidelines

- Use simple, recognizable shapes
- Ensure visibility at small sizes (16x16)
- Use a consistent color scheme matching the app's theme
- Consider light and dark backgrounds for system tray icons
